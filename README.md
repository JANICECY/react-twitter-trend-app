# React-twitter-trend-app
This app is able to get latest twitter trends by using APIs provided by [Twitter](https://developer.twitter.com/en/docs).
### version 1
- [x] Get trends by country 
- [ ] Get trends by keyword 
## Built With
- React
- react-redux
- react-thunk
- semantic-ui-react
- axios
- Flask
